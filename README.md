The svrx plugin for less

## Usage

> Please make sure that you have installed [svrx](https://svrx.io/) already.

### Via CLI

```bash
svrx -p less
```

### Via API

```js
const svrx = require('@svrx/svrx');

svrx({ plugins: [ 'less' ] }).start();
```
